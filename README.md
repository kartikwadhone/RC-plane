#  Quick Start

To build this RC plane:

1. Download the files from this repository.
2. Print only the STL files that are supposed to be printed
3. Assemble the airframe.
4. Install the electronics.
5. Balance the center of gravity.
6. Fly!

---

#  Features

- Over 15 min Flight time
- Under 800g (Depends on configuration)
- Highly Stable
- Good Vertical Climb ability
- Optional Landing Gear (With Front Suspention!!)
